<h1>Layout 3</h1>
<br>
<p>O menu de navegação está abaixo do cabeçalho alinhado ao centro. O aside com o conteúdo relacionado fica à direita do main que tem o conteúdo principal do site. Quando apresentado em telas menores, como smartphones, o menu fica oculto, sendo exibido apenas quando o botão de menu no canto superior direito é clicado.</p>
